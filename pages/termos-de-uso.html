Termos de Uso

Estes Termos de Uso regulam o acesso e utilização da Área do Cliente do sistema CASEG Protege. Ao utilizar este sistema, o usuário declara estar ciente e de acordo com as condições aqui estabelecidas.

1. Finalidade do Sistema

O CASEG Protege disponibiliza uma área restrita para clientes, permitindo acesso a documentos, comunicados e informações relacionadas aos serviços prestados pela empresa.

2. Acesso e Responsabilidade do Usuário

O acesso ao sistema é realizado mediante autenticação com CPF/CNPJ e senha pessoal. O usuário é responsável por manter a confidencialidade de suas credenciais de acesso, não devendo compartilhá-las com terceiros.

Qualquer atividade realizada com as credenciais do usuário será de sua responsabilidade.

3. Uso Adequado do Sistema

O usuário compromete-se a utilizar o sistema apenas para os fins a que se destina, sendo proibido:

• Tentar acessar dados de outros usuários
• Utilizar o sistema para fins ilícitos
• Realizar qualquer tentativa de violação de segurança

4. Disponibilidade do Sistema

O CASEG Protege poderá passar por manutenções periódicas ou eventuais indisponibilidades, não sendo garantido acesso ininterrupto ao sistema.

5. Documentos Disponibilizados

Os documentos disponibilizados na Área do Cliente são de caráter confidencial e destinados exclusivamente ao cliente vinculado à conta.

É proibida a divulgação, compartilhamento ou uso indevido desses documentos sem autorização.

6. Proteção de Dados

O tratamento de dados pessoais é realizado conforme a legislação vigente, especialmente a Lei Geral de Proteção de Dados (LGPD).

Para mais informações, consulte a nossa Política de Privacidade.

7. Alterações nos Termos

O CASEG Protege poderá atualizar estes Termos de Uso a qualquer momento, sendo recomendada a revisão periódica deste documento.

8. Contato

Em caso de dúvidas sobre estes Termos, entre em contato pelo e-mail: casegprotege@hotmail.com
